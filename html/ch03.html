문서 구조화

시맨틱 태그

사람이 태그를 보고 잘 이해할 수 있도록 구조화

  • header: 머리말
  • section: 장 혹은 절
  • article: 작은 기사
  • aside: 부수적인 글
  • nav: 하이퍼링크 모음
  • footer: 바닥글

class와 id의 전역속성

  • class: 같은 종류의 태그를 하나로 묶음, 같은 class 여러번 사용 가능
  • id: 다른 태그와 구분, id 속성값은 유일해야 함.
  • title: 툴팁을 표시

시맨틱 블록 태그와 시맨틱 인라인 태그

block element는 페이지의 전체 row를 차지, 크기 조정 가능

inline element는 내용이 존재하는 부분만 차지

inline block은 inline이지만 크기 조정 가능

시맨틱 블록 태그

  • figure: 이미지를 블록화, 이미지 제목은 figcaption으로 작성
  • details: 상세정보를 숨기거나 감추는 태그

시맨틱 인라인 태그

  • ruby: 루비 첨자를 만드는 태그, 첨자 내용은 rt태그에 작성
  • time: 시간 표시
  • meter: 값을 막대 정보로 표시
  • progress: 작업의 진행 정도를 표시

웹 폼 만들기

입력관련 주요 태그

입력 태그 input과 주요 태그

input태그의 type속성값

  • text: 한 줄 입력
  • password
  • button, submit, reset, image: 단순 버튼, 제출 버튼, 리셋 버튼, 이미지 버튼
  • month, week, date, time
  • datetime-local: 로컬 날짜와 시간 입력
  • number, range: 숫자 입력, 슬라이드 바를 통한 숫자 입력
  • radio: 단수 선택 입력
  • checkbox: 복수 선택 입력
  • color: 색상 입력
  • file: 파일 업로드
  • email, tel, url, research: 이메일, 전화, url, 검색어 입력

type=text or password일때 공통 사용 속성

  • name="입력 요소 이름"
  • maxlength="최대 입력 문자 수"
  • size="문자 창의 크기 (문자 수)"
  • value="초기 문자"
  • placeholder="안내 문자"

문자 여러 줄 입력 textarea

textarea의 속성

  • rows="창의 행 크기 (문자 수)”
  • cols="창의 열 크기 (문자 수)”
  • wrap="soft": 입력 내용 중 enter표시 방법
  • placeholder="안내 문자"

옵션이 있는 입력

  • datalist: 입력 문자의 옵션 제공
  • select: 리스트를 가진 콤보 박스
  • : 단순 버튼
  • select 태그
  • input type=radio, checkbox

입력 태그 input과 문자 외 데이터 입력

  • number: 숫자를 입력받는 속성값
  • range: 슬라이드 바로 숫자를 입력받는 속성값

웹 폼을 담는 상자

form 태그의 속성

  • name="웹 폼 이름"
  • action="서버 응용 프로그램의 url"
  • enctype="데이터 암호화 방식"
  • method="get or post"
  • target="윈도우 이름"
