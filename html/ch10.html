  • document
  • navigator
  • screen
  • history
  • location

  • window

window 객체 사용하기

윈도우 열고 닫기

  • 윈도우 열기 open()
window.open(URL, Name, Feature)

웹 페이지 표시 위치 지정 Name 값
-
_self: 현재 창에서 보여줌
-
_blank: 새 창에서 보여줌
-
_parent: 부모 창에서 보여줌. 부모 창이 없으면 _self와 동일
-
_top: 최상단에서 보여줌. 최상단이 없으면 _self와 동일

window 객체 속성 지정 Feature 값
- width
- height
- left
- top
- resizable
- location
- scrollbars
- status
- toolbar

반응형 웹 만들기

반응형 웹

  • landscape & portrait
  • 반응형 웹 디자인
  • 뷰포트(viewport)
<meta name="viewport" content="width=device-width, initial-scale=1.0">

반응형 웹 페이지와 미디어 쿼리

  • viewport
@media screen and (orientation: portrait) {
  body {font-size: 12px;}
}

@media 표현식의 속성
- width
- height
- device-width
- device-height
- orientation

  • @media (width: 700px): 폭이 700px일 때
  • @media (min-width: 700px): 폭이 700px보다 클 때
  • @media (max-width: 700px): 폭이 700px보다 작을 때
  • @media (min-width: 700px) and (max-width: 900px): 폭이 700보다 크고 900보다 작을 때

  • vw: 뷰포트 폭의 1/100
  • vh: 뷰포트 높이의 1/100
  • vmax: vw, vh 중 큰 값
  • vmin: vw, vh 중 작은 값
