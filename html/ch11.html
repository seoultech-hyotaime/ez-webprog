캔버스로 그림 그리기

캔버스 기본 기능

<canvas> 태그 사용하기

문서에 캔버스 추가하기

<canvas id="myCanvas" width="600" height="400"></canvas>
const canvas = document.getElementById("myCanvas");const ctx = canvas.getContext("2d");
  • 책을 채운 도형 그리기
ctx.fillStyle = "blue"; // 사각형의 안쪽 색상ctx.fillRect(20, 20, 50, 50) // (X, Y, width, height)
  • 테두리만 있는 도형 그리기
ctx.strokeStyle = "red"; // 사각형의 테두리 색상ctx.strokeRect(20, 20, 50, 50); // 테두리만 있는 사각형
  • 그래픽 지우기
ctx.clearRect(0, 0, canvas.width, canvas.height) // 화면 전체 지우기requestAnimationFrame(draw) // draw함수를 작동 시키기

ctx.closePath(); // 경로를 닫음. 종료점과 시작점을 연결
