배열과 Array 객체

let arr1 = [1, 2, 3, 4, 5];let arr2 = [1, "첫번째", 2.34, true];

Array 객체 메서드

  • arr1.concat(arr2): arr1 + arr2를 리턴
  • filter(function()): function의 조건에 맞는 배열의 복사본 리턴
  • indexOf(x, i): 인덱스 i부터 시작하여 처음 x가 나타나는 인덱스를 찾아 리턴, 없으면 -1
  • join(s): 배열의 모든 문자열을 합친 문자열 반환, 사이에 s삽입
  • pop()
  • push(x)
  • reverse()
  • shift(): =popleft
  • slice(i, j): =str[i:j]
  • sort()
  • toString()
  • unshift(x): =appendleft(x)

String 객체 메서드

  • charAt(i): 인덱스 i의 문자 리턴
  • replace(s, t): 문자 s를 문자 t로 변경
  • search(s)
  • split(s, limit): s를 분리자로 하여 문자열 분리하고 배열 리턴, limit은 반환되는 배열의 크기 제한
  • substr(i, len): 인덱스 i부터 len만큼의 문자열을 배열로 리턴
  • substring(i, j)
  • toLowerCase()
  • toUpperCase()
  • trim(): =strip(), 양끝의 공백 문자 제거

Date 객체

  • get/setFullYear(): 4자리 연도
  • get/setMonth(): 월(0 ~ 11), 출력 시 +1, 입력 시 -1
  • get/setDate(): 일(1 ~ 31)
  • get/setDay(): 요일(0 ~ 6)
  • get/setHours(): 시(0 ~ 23)
  • get/setMinutes(): 분(0 ~ 59)
  • get/setSeconds(): 초(0 ~ 59)
  • get/setMilliseconds(): 밀리초(0 ~ 999)
  • toLocaleString(): 연, 월, 일, 시간 표시

function으로 객체 선언하기

function Person (name, age) {
  this.name = name;  this.age = age;  this.intro = function () {
    document.write("My name: " + this.name + ", age: ", this.age + "<br>");  }
};let person1 = new Person("cho", 23);let person2 - new Person("kim", 24);person1.intro();person2.intro();

class를 이용한 객체 상속

ES6에 class가 추가되었지만 실제 동작은 prototype으로 변환하여 작동함

class Person {
  constructor(name, age) {
    this.name = name;    this.age = age;  }
  intro() {
    document.write("My name: " + this.name + ", age: ", this.age + "<br>");  }
}
let person1 = new Person("cho", 23);person1.intro()

상속은 extends로 확장

super를 이용해 부모 클래스의 속성에 접근

class Animal {
  constructor(name, age) {
    this.name = name;    this.age = age;  }
}
class Person extends Animal {
  constructor(name, age, id) {
    super(name, age) // Animal 객체의 속성에 접근    /*    super(name, age) 대신    super()    this.name = name;    this.age = age;    로 사용해도 무관    */    this.id = id;  }
}
